## Custom Settings

Other background colors maybe you like:

- #fffae8
- #fffffe
- #fdf6e3
- #fffcf8
- #f8f8f8
- #fef8ec

```json
"workbench.colorCustomizations": {
  "[Relax Light]": {
    "editor.background": "#fffae8"
  }
}
```
